# PROYECTO: CONVERSOR UNIVERSAL

## ¿Para qué servirá?
El conversor servirá para hacer conversiones de unidades, principalmente de temperatura.

## ¿Cómo servirá?
Tendrá paneles con opciones de conversión. También incluirá historial de conversiones y referencias para entender mejor cada unidad.
